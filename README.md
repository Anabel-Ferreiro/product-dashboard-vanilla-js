# Product Dashboard
Coding Project 2 - Product Dashboard Setup Using JavaScript and the Store Products API

## Description
A product dashboard built with HTML, CSS, and JavaScript that fetches and displays product data from an external API using both Promise-based fetch and async/await.

## How to Run
1. Open index.html in a browser
2. View the product dashboard
3. Hover over product cards to see the effect
